# Nervesystemet og hormonsystemet Quiz

En interaktiv quiz om samarbeidet mellom nervesystemet og hormonsystemet i naturfag.

## Innhold

Quizen dekker følgende temaer:

1. **Melatonin og døgnrytme** - Hvordan lys påvirker melatoninproduksjon og søvn
2. **Temperaturregulering** - Samarbeid mellom nervesystem og hormonsystem for å regulere kroppstemperatur
3. **Stress og adrenalin** - Hvordan kroppen reagerer på stress
4. **Sult og metthet** - Hormonregulering av appetitt med ghrelin og leptin
5. **Blodsukkerregulering** - Insulin og glukagon i regulering av blodsukker
6. **Pubertet** - Hormonell kontroll av pubertetsutviklingen

## Antall spørsmål

Quizen inneholder totalt **60 spørsmål** fordelt på de seks temaene.

## Funksjonalitet

- Spørsmålene blandes tilfeldig ved hver gjennomgang
- Viser umiddelbar feedback på svar
- Forklaringer ved feil svar
- Poengsum og prosent ved fullføring
- Mulighet til å starte på nytt

## Bruk

Åpne `index.html` i en nettleser for å starte quizen.
